Architecture & Roadmap

How we get from here to playable

Fork First, Extract Later

Phase 1 Full copy of shader-playground → spanda/ repo. Messy and fast. Phase 2 Extract shared code → flame-engine library crate. Clean APIs. Phase 3 Platform targets: web (wasm), mobile (iOS/Android). Engine stays pure.

M0: Fork & Fly

~1 week

Copy shader-playground. Strip dev tooling. Add a camera you can move through the fractal. Doesn't need to be a "game" — just prove you can fly through flame space and it feels like something.
Deliverable: You press WASD and drift through a fractal flame. That's it.

M1: Three Modes

~2 weeks

Implement drift, tunnel, and gestural input modes. Each is a different way to move through the same world. Play all three with music running. Find the magic.
Deliverable: Press 1/2/3 to switch modes. One of them makes you go "oh HELL yes."

M2: The World Evolves

~2 weeks

Movement becomes selection pressure. Multiple genome regions in the world. Flying toward beauty → more of it. Beat-quantized mutations. Music drives evolution. R&D spike on multi-flame rendering happens here.
Deliverable: The world visibly responds to where you go and what's playing.

M3: The World Learns You

~2 weeks

Taste model wired to implicit behavior signals. R&D spike on signal filtering. Over a 10-minute session, your Spanda should feel distinctly yours. Game modes: zen (default), challenge (optional scoring), garden (sandbox).
Deliverable: Play for 10 minutes. The world at the end looks nothing like the start — and it's yours.

M4: Engine Extraction

~2 weeks

By now we know what the game IS. Extract flame-engine as a shared library crate. Cargo workspace. Clean API boundaries. shader-playground and Spanda both depend on it.
Deliverable: Two binaries, one engine. Both build and run.

M5: Silent Communion

~3-4 weeks

Multiplayer. Server architecture. Anonymous presence. Genome cross-pollination. Session-end reveal. Security hardening. Subscription payment integration.
Deliverable: Two players drift together. Their flames touch. Something new is born.

M6: Release

~2 weeks

Polish. Music file loading. Platform packaging (Steam, itch.io). Web build (wasm/WebGPU). Trailer. Launch.
Deliverable: People play Spanda. And it's wonderful.
